DecoderвҖӢ

Decoderзҡ„д»»еҠЎжҳҜз”ҹжҲҗиҫ“еҮәпјҢеҸҜд»Ҙж №жҚ®жҳҜеҗҰдёҖж¬ЎжҖ§з”ҹжҲҗиҫ“еҮәеҲҶдёәAutoregressiveпјҲиҮӘеӣһеҪ’пјҢabbr. ARпјүд»ҘеҸҠNon-AutoregressiveпјҲйқһиҮӘеӣһеҪ’пјҢabbr. NARпјүдёӨз§ҚжЁЎејҸгҖӮ

иҮӘеӣһеҪ’зұ»еһӢзҡ„DecoderйңҖиҰҒйҖҗжӯҘз”ҹжҲҗиҫ“еҮәпјҢе№¶е°Ҷд№ӢеүҚиҮӘиә«иҫ“еҮәзҡ„жүҖжңүиҜҚжұҮз»ҸиҝҮеөҢе…ҘеұӮеҗҺз”ҹжҲҗtokenдҪңдёәдёӢдёҖж¬Ўзҡ„иҫ“е…ҘпјҢйҖҡеёёжҜҸж¬Ўз”ҹжҲҗдёҖдёӘиҜҚжҲ–дёҖдёӘз¬ҰеҸ·гҖӮиҝҷз§Қж–№ејҸзҡ„зјәзӮ№жҳҜйңҖиҰҒдҝқеӯҳе’Ңжӣҙж–°иҜҚиЎЁдёӯзҡ„жүҖжңүеҸҜиғҪйҖүйЎ№пјҢеӣ жӯӨеңЁеӨ§иҜҚжұҮиЎЁдёҠеҸҜиғҪдјҡеҸҳеҫ—йқһеёёж…ўгҖӮз„¶иҖҢпјҢе®ғзҡ„дјҳзӮ№жҳҜиғҪеӨҹеҲ©з”ЁдёҠдёӢж–ҮдҝЎжҒҜжқҘз”ҹжҲҗиҫ“еҮәпјҢиҝҷжңүеҠ©дәҺжҸҗй«ҳзҝ»иҜ‘зҡ„иҙЁйҮҸгҖӮ

йқһиҮӘеӣһеҪ’зұ»еһӢзҡ„DecoderиҜ•еӣҫеңЁдёҖж¬Ўж“ҚдҪңдёӯз”ҹжҲҗж•ҙдёӘиҫ“еҮәеәҸеҲ—гҖӮиҝҷйҖҡеёёйҖҡиҝҮдҪҝз”ЁиҜёеҰӮжіЁж„ҸеҠӣжңәеҲ¶зӯүзӯ–з•ҘжқҘе®һзҺ°пјҢиҝҷдәӣзӯ–з•Ҙе…Ғи®ёи§Јз ҒеҷЁе…іжіЁиҫ“е…ҘеәҸеҲ—зҡ„дёҚеҗҢйғЁеҲҶпјҢеҗҢж—¶з”ҹжҲҗиҫ“еҮәеәҸеҲ—зҡ„дёҚеҗҢйғЁеҲҶгҖӮNARзҡ„дјҳзӮ№еңЁдәҺе…¶й«ҳж•ҲжҖ§пјҢеӣ дёәе®ғдёҚйңҖиҰҒдҝқеӯҳе’Ңжӣҙж–°еӨ§йҮҸзҡ„еҸҜиғҪйҖүйЎ№гҖӮз„¶иҖҢпјҢз”ұдәҺе®ғдёҚиғҪеҲ©з”ЁдёҠдёӢж–ҮдҝЎжҒҜжқҘз”ҹжҲҗиҫ“еҮәпјҢеӣ жӯӨе…¶з”ҹжҲҗзҡ„иҫ“еҮәиҙЁйҮҸжҷ®йҒҚдјҡдҪҺдәҺARгҖӮ

иҜҚжұҮиЎЁпјҲVocabularyпјүвҖӢ

иҜҚжұҮиЎЁпјҲVocabularyпјүжҳҜдёҖдёӘеҢ…еҗ«дәҶеңЁзү№е®ҡиҜӯиЁҖжҲ–д»»еҠЎдёӯжүҖжңүеҸҜиғҪеҮәзҺ°зҡ„жүҖжңүеҚ•иҜҚжҲ–ж Үи®°зҡ„йӣҶеҗҲгҖӮеңЁиҮӘз„¶иҜӯиЁҖеӨ„зҗҶпјҲNLPпјүдёӯпјҢиҜҚжұҮиЎЁжҳҜи®ӯз»ғжЁЎеһӢж—¶жүҖдҪҝз”Ёзҡ„е”ҜдёҖеҚ•иҜҚзҡ„йӣҶеҗҲпјҢз”ұе…·дҪ“зҡ„з”ҹжҲҗд»»еҠЎиҖҢзЎ®е®ҡгҖӮ

DecoderжҜҸдёҖжӯҘзҡ„иҫ“еҮәжҳҜдёҖдёӘз»ҸиҝҮSoftmaxзҡ„Probability DistributionпјҲжҰӮзҺҮеҲҶеёғпјүпјҢд»ЈиЎЁзқҖиҜҚжұҮиЎЁдёӯжҜҸдёҖдёӘиҜҚжұҮеҪ“еүҚз”ҹжҲҗзҡ„жҰӮзҺҮпјҢеҸ–жңҖеӨ§жҰӮзҺҮеҖјзҡ„иҜҚжұҮдҫҝжҳҜжЁЎеһӢеҪ“еүҚж—¶й—ҙжӯҘиҫ“еҮәзҡ„иҜҚжұҮгҖӮ

Beginз¬ҰеҸ·вҖӢ

и§Јз ҒеҷЁпјҲDecoderпјүеңЁжҜҸдёӘж—¶й—ҙжӯҘпјҲжҲ–жҜҸдёӘи§Јз ҒжӯҘйӘӨпјүзҡ„иҫ“е…ҘйғҪжқҘиҮӘдәҺеүҚдёҖдёӘж—¶й—ҙжӯҘиҮӘиә«зҡ„иҫ“еҮәд»ҘеҸҠзј–з ҒеҷЁпјҲEncoderпјүзҡ„иҫ“еҮәгҖӮзү№еҲ«ең°пјҢйҰ–дёӘж—¶й—ҙжӯҘзҡ„иҫ“е…ҘжҳҜBeginз¬ҰеҸ·д»ҘеҸҠзј–з ҒеҷЁпјҲEncoderпјүзҡ„иҫ“еҮәпјҢеңЁжҜҸдёӘеҗҺз»ӯзҡ„ж—¶й—ҙжӯҘпјҢи§Јз ҒеҷЁзҡ„иҫ“е…ҘдјҡжҳҜеүҚдёҖдёӘж—¶й—ҙжӯҘиҮӘиә«зҡ„иҫ“еҮәд»ҘеҸҠзј–з ҒеҷЁпјҲEncoderпјүзҡ„иҫ“еҮәпјҢзӣҙеҲ°з”ҹжҲҗеәҸеҲ—зҡ„з»“жқҹгҖӮ

жҸҗзӨә

Beginз¬ҰеҸ·жҳҜеңЁLexiconдёӯж·»еҠ зҡ„зү№ж®Ҡз¬ҰеҸ·пјҢз”ЁжқҘиЎЁзӨәDecoderз”ҹжҲҗзҡ„ејҖе§ӢгҖӮBeginз¬ҰеҸ·йҖҡеёёиў«еөҢе…ҘеҲ°дёҖдёӘдҪҺз»ҙзҡ„иҝһз»ӯеҗ‘йҮҸз©әй—ҙдёӯпјҢиҝҷдёӘеҗ‘йҮҸз©әй—ҙжҳҜйҖҡиҝҮеөҢе…ҘеұӮпјҲEmbedding LayerпјүеӯҰд№ еҫ—еҲ°зҡ„пјҢеңЁеөҢе…ҘеұӮдёӯпјҢзҰ»ж•Јзҡ„з¬ҰеҸ·иў«жҳ е°„еҲ°дёҖдёӘе®һж•°еҗ‘йҮҸгҖӮ

Beginз¬ҰеҸ·еҸҲеҸ«Startз¬ҰеҸ·жҲ–SOSз¬ҰеҸ·пјҲStart Of SentenceпјүпјҢйғҪжҳҜиЎЁзӨәз”ҹжҲҗзҡ„ејҖе§ӢгҖӮEndз¬ҰеҸ·еҸҲеҸ«EOSз¬ҰеҸ·пјҲEnd Of SentenceпјүгҖӮ

Endз¬ҰеҸ·вҖӢ

еңЁDecoderзҡ„з”ҹжҲҗдёӯпјҢжҜҸдёҖдёӘж—¶й—ҙжӯҘзҡ„иҫ“еҮәжҳҜиҜҚжұҮиЎЁдёӯжҜҸдёҖдёӘеҚ•иҜҚз»ҸиҝҮSoftmaxд№ӢеҗҺзҡ„жҰӮзҺҮеҲҶеёғгҖӮдёәдәҶдҝқиҜҒз”ҹжҲҗд»»еҠЎеҸҜд»ҘйҖҡиҝҮжЁЎеһӢиҮӘе·ұеҒңжӯўиҖҢдёҚжҳҜдёҖзӣҙйҮҚеӨҚпјҢжҲ‘д»¬еҗ‘Decoderзҡ„иҫ“еҮәдёӯеҠ е…ҘEndз¬ҰеҸ·зҡ„з”ҹжҲҗпјҢеҚіжҜҸдёҖж¬Ўиҫ“еҮәйҷӨдәҶиҜҚжұҮиЎЁзҡ„жүҖжңүиҜҚжұҮеӨ–иҝҳжңүEndз¬ҰеҸ·зҡ„жҰӮзҺҮпјҢеҪ“Endз¬ҰеҸ·жҳҜеңЁжүҖжңүиҜҚжұҮдёӯжҰӮзҺҮжңҖеӨ§зҡ„иҜҚжұҮж—¶пјҢз”ҹжҲҗеҒңжӯўгҖӮ
